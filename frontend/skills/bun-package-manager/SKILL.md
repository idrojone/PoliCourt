# Entorno: Bun

Este proyecto utiliza **Bun** como entorno de ejecución y gestor de paquetes.
- Para instalar dependencias: usa `bun add <paquete>` (nunca `npm install`).
- Para ejecutar scripts: usa `bun run <script>` o `bun dev`.
- Si necesitas añadir un componente de shadcn, asume el comando `bunx --bun shadcn-ui@latest add <componente>`.
